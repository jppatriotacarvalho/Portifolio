Meus Projetos

Bem-vindo à minha galeria de projetos! Aqui você encontrará uma seleção dos trabalhos que desenvolvi durante minha jornada acadêmica, com foco especial em jogos educativos e acessíveis. Cada projeto representa um desafio único e uma oportunidade de aprendizado.

BrainPop!

Através de pesquisas, descobrimos barreiras no aprendizado de crianças com deficiência visual. Isso nos motivou a criar o BrainPop!, um jogo educativo para estimular a memória e o raciocínio lógico de forma acessível.

O jogo usa recursos como braile, sons e áudios descritivos. O jogador passa por um ciclo de "jogo da memória" sonoro e, ao acertar, responde a uma pergunta da matéria escolar escolhida, vencendo após 10 acertos.

Não posso deixar de destacar a visita ao Instituto de Cegos Antônio Pessoa de Queiroz, onde o potencial do jogo para o desenvolvimento e inclusão foi validado.

Tecnologias e Habilidades:

Design Inclusivo Acessibilidade Braile Hardware Sons

Colaboradores: Arthur da Fonte de Oliveira, Walter Maia, Filipe Santos, Bernardo, Lucas, Gabriel Ribeiro, Giovana, Manuela e Antônio.

Missão no Fundo do Mar

Desenvolvido para a disciplina de Algoritmos e Estrutura de Dados, este jogo coloca o jogador no controle de um submarino para resgatar tesouros no oceano.

O projeto foi uma aplicação prática de conceitos complexos, utilizando lista encadeada para coletáveis, algoritmos de ordenação para o ranking, fila para a ordem de jogadores e árvore binária de busca para estruturar as pontuações. Além disso, implementamos uma IA para dar dicas durante o jogo e utilizamos a biblioteca gráfica Raylib para criar a interface.

Tecnologias e Habilidades:

C Raylib Listas Encadeadas Filas Árvores Binárias Algoritmos de Ordenação

Colaboradores: Gustavo Carneiro, Mateus Dornellas.

The Legend of Erick

Criado para a disciplina de Introdução à Computação, este projeto foi um desafio pessoal, pois exigiu o aprendizado rápido de uma plataforma No-Code para desenvolvimento de jogos.

Inspirado em jogos de puzzle, o "The Legend of Erick" foi desenvolvido em uma semana de aprendizado intenso, resultando em um jogo funcional e uma apresentação completa do projeto.
